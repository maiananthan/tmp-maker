# Temp Note Maker

- This plugin is to create a temporary files based on the date for temporary use. Also, cleans up temporary files when it's past the retention days during start-up and through command palette.

## INSTALL

- This plugin can be installed via the Community Plugins tab within Obsidian. Search for "Temp Note Maker"

## USAGE

- Click on the left side icon to create a new temporary note with today's date

## CUSTOMIZATION

| Option | Description | Default Value | Options Available |
|:---:|:---:|:---:|:---:|
|Temp Folder| Name of the folder under which notes are created| `tmp` | |
|Retention days| Notes which are older than mentioned will be auto removed | `0` | 0 (disable), +ve |
|Auto-cleanup on startup| Auto clean up notes during start up| | |
